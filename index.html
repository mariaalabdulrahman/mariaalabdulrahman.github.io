Pothole Severity Classification

Instance Segmentation Methods - Decision Trees

Pothole Severity Classification - 2023

Images depicting potholes were collected (17400 images).

An instance segmentation model was trained (YoloV8). Structural features such as pothole area, height, and width were extracted from the segmented images

A decision tree classifier is trained on the extracted features; highest accuracy yeilded was 98.9%.
